We will use smtp module here.

  • Step 1: First create an smtp object with sender's mail and default value 587.
  • Step 2: Start the tls protocol
  • Step 3: For authentication we have to login to our gmail account, for that we'll use login function.
  • Step 4: Sending mail
import smtplib
s=smtplib.SMTP('sender_mail',587)
#start tls
s.starttls()
# authenticate
s.login('sender_mail','sender_password')
message=input('hello')
s.sendmail('sender_mail','reciever_mail',message)
print('Mail send')
s.quit()
